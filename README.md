# Habit-Tracker

A premium Habit Tracker Web Application built using **HTML, CSS, JavaScript, and Chart.js**.

Inspired by modern productivity tools like **Notion, Habitify, TickTick, Loop Habit Tracker, and Linear**, Habit tracker helps users build consistency, track habits, manage goals, maintain daily journals, and visualize progress through analytics.

---

## Features

### Habit Tracking

* Create, edit, and delete habits
* Mark habits as completed daily
* Automatic streak calculation
* Habit completion percentage
* Persistent habit history

### Goals Management

#### Monthly Goals

* Add and manage monthly goals
* Progress tracking
* Animated progress bars

#### Weekly Goals

* Add and manage weekly goals
* Completion tracking
* Weekly summaries

### Daily Journal

* Date-based journal entries
* Save reflections, notes, and achievements
* Automatically linked to selected calendar dates
* Persistent storage

### Interactive Calendar

* Monthly calendar view
* Previous and next month navigation
* Daily completion percentages
* Historical habit tracking
* Journal integration

### Analytics Dashboard

* Habit completion trends
* Weekly consistency tracking
* Monthly performance visualization
* Most consistent habit detection
* Goal completion statistics
* Current and longest streak tracking

### User Experience

* Modern SaaS-inspired design
* Fully responsive layout
* Dark and Light mode
* Glassmorphism UI
* Smooth animations
* Toast notifications
* Modal dialogs
* Local Storage persistence

## Data Storage

All user data is stored locally in the browser using Local Storage.

Stored data includes:

* Habits
* Goal Progress
* Journal Entries
* Theme Preferences
* Analytics Data

No external database is required.
